# Software-Renderer
### What is included ?
   - Perspective correct texture mapping.
   - OBJ Models Support.
   - Lighting (Phong Shading).
   - SSE Wrapper (Most Maths calculations in SIMD instructions).
   - Drawing edge to edge using Fixed Point arithmetic to save up cycles.
